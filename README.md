## 使用技術

- HTML5（語意化結構）
- CSS3（Flexbox / Grid 彈性佈局、CSS 變數）
- RWD（Media Queries）
- JavaScript（返回頂部功能）

## 功能介紹

- 響應式排版，支援手機、平板與桌機瀏覽
- 自訂 CSS 變數，統一管理色彩與字體大小
- Back to Top 置頂按鈕
- SEO 友善語意標籤
- 平滑滾動（Smooth Scroll）功能
